Detailed Description

A joystick class to read one joystick via ADC and GPIO input.

Class can be used to setup a joystick on a Nucleo l476RG board.

def Joystick_Driver.Joystick.toggle_pen (   self)

Set an LED on or off to indicate toggle of the joystick switch.

Uses a digital pin of the L476RG to turn an LED on or off with a toggle logic control. This was useful for testing use the switch of the joystick to toggle up and down the pen actuator of the plotter in manual mode.

Reads the X-position potentiometer of the joystick.

Uses the 12-bit ADC of the L476RG to read the X-position potentiometer of the joystick. A conditional handles any bias from zero position of the potentiometer when released.
